About Me

Projects

CSCI 62 - Advanced
Programming Final Project:
Social Network Project

I built a social networking application in C++ using Qt Creator, featuring an interactive interface with buttons, text fields, lists, and a search bar. The project included user authentication, account creation, and customizable privacy settings for posts and profiles. I also implemented file-based data storage to handle friend requests, post creation and likes, while ensuring unique usernames and secure user management.

Independent Project:
Trip Planner

I developed a trip-planning application in Python with SQL integration to help organize study-abroad travel in Europe. The program allows users to input trip details such as dates, activities, transportation methods, and time allocations. It also includes a built-in currency converter (EUR/GBP → USD) for budgeting and expense tracking, and generates a complete, printable itinerary for easy access while traveling.

P&M Caseworks:
Website

I collaborated with the owners of P&M Caseworks to design and develop a responsive multi-page website showcasing their services. Using Figma for mockups and React, HTML, CSS, and JavaScript for implementation, I ensured mobile-friendly performance and security. After resolving vulnerabilities, I configured a new AWS domain. This project is still a work in progress.
